Icons for music player are from https://design.google.com/icons/ which is open source

Jaudiotagger is used to read song metadata. Link: http://www.jthink.net/jaudiotagger/

Code for rating scale conversions in Song.java taken from https://github.com/soc/jaudiotagger/blob/master/src/main/java/org/jaudiotagger/tag/id3/reference/MediaMonkeyPlayerRating.java

Alert and dialog boxes referred from http://code.makery.ch/blog/javafx-dialogs-official/

Jpathwatch is used to monitor the file system changes. Link: https://jpathwatch.wordpress.com/

Java Native Access used to move file to recycle bin. Link: https://github.com/java-native-access/jna
